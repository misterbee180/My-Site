Combinding the tiled image with the background color creates the effect you see here.

It's also worth nothing what I'm starting to realize about image types:
  • Jpg - a simple and small image format
  • Gif - another small image format that allows for empty space
  • Png - a larger image format that allows for empty space and alpha color levels
  • Gimp - an image program that sucks
